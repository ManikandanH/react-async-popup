Welcome to [react-async-popup](https://github.com/aadilhasan/react-async-popup), an accessible promise based popup library for [react](https://reactjs.org/), which helps to reduce extra state management needed for Popups.
It is inspired from native [confirm](https://developer.mozilla.org/en-US/docs/Web/API/Window/confirm) method.

## Installation

To install the stable version you can use npm or yarn:

```bash
npm install --save react-async-popup
```

or

```bash
yarn add react-async-popup
```
